# Chrome 擴充應用程式測試指南

## 快速測試步驟

### 1. 構建擴充應用程式
```bash
npm run build:extension
```

### 2. 在 Chrome 中載入擴充應用程式
1. 打開 Chrome 瀏覽器
2. 前往 `chrome://extensions/`（或右鍵點擊擴充應用程式圖示 → 管理擴充功能）
3. 開啟右上角的「**開發人員模式**」開關
4. 點擊「**載入未封裝項目**」按鈕
5. 選擇專案中的 `dist` 資料夾（不是根目錄，是 dist 資料夾）
6. 應該會看到擴充應用程式出現在列表中

### 3. 測試擴充應用程式
1. 點擊 Chrome 工具列上的擴充應用程式圖示（如果沒有看到，點擊拼圖圖示找到它）
2. 彈出視窗應該會打開（寬度約 1200px，高度約 800px）
3. 測試各項功能：
   - 填寫表單資訊
   - 上傳截圖
   - 繪製標記框
   - 下載測試報告

### 4. 查看錯誤（如果有問題）
1. 在擴充應用程式管理頁面，點擊擴充應用程式下方的「**檢查檢視服務**」
2. 或右鍵點擊彈出視窗 → 「**檢查**」
3. 查看 Console 標籤的錯誤訊息

## 常見問題

### 擴充應用程式無法載入
- ✅ 確認已執行 `npm run build:extension`
- ✅ 確認載入的是 `dist` 資料夾（不是根目錄）
- ✅ 檢查 `dist` 資料夾中是否有 `manifest.json` 和 `popup.html`

### 彈出視窗是空白
- 打開開發者工具檢查錯誤
- 確認 `dist/assets/` 資料夾中有 JavaScript 文件
- 確認 `dist/popup.html` 中的腳本路徑正確

### 功能無法使用
- 檢查 Chrome 擴充應用程式管理頁面的權限設定
- 確認已授予必要的權限（downloads）

## 重新構建

每次修改程式碼後，需要重新構建：
```bash
npm run build:extension
```

然後在 Chrome 擴充應用程式管理頁面點擊擴充應用程式的「重新載入」按鈕（🔄 圖示）。
