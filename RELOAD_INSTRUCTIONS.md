# 重新載入擴充應用程式步驟

如果遇到「無法載入擴充功能」錯誤，請按照以下步驟：

## 方法 1：完全移除並重新載入（推薦）

1. **打開 Chrome 擴充應用程式管理頁面**
   - 前往 `chrome://extensions/`
   - 或右鍵點擊擴充應用程式圖示 → 管理擴充功能

2. **移除擴充應用程式**
   - 找到「QA 測試回報產生器」
   - 點擊「移除」按鈕（垃圾桶圖示）
   - 確認移除

3. **重新載入**
   - 點擊「載入未封裝項目」
   - 選擇專案中的 **`dist` 資料夾**（不是根目錄！）
   - 確認載入

## 方法 2：重新構建後再載入

1. **確保已重新構建**
   ```bash
   npm run build:extension
   ```

2. **移除並重新載入**
   - 按照方法 1 的步驟操作

## 驗證文件

確保 `dist` 資料夾中包含：
- ✅ `manifest.json`
- ✅ `background.js`
- ✅ `popup.html`
- ✅ `assets/` 資料夾（包含 JS 和 CSS 文件）

如果缺少任何文件，請重新構建。
